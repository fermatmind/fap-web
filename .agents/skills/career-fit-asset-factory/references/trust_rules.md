# Career Fit Trust Rules

Block diagnosis, destiny, income, mental-health, employment, or personal-value claims. Block claims that a type guarantees fit.

Trust audit must fail rows with fabricated facts, unsupported proxy use, raw internal metadata, missing source boundaries, or reader-facing claims that exceed the evidence.

Trust audit must also fail a Big Five bridge row when:

- publication is inferred from primary-record existence, import success, route
  HTTP 200, sitemap presence, or a working-revision pointer;
- a draft/review/working revision or generated package is used as public truth;
- published projection identity, locale, or claim permissions cannot be proven;
- Big Five is used to rank occupations or replace RIASEC/work-activity evidence;
- private assessment or commerce data enters the public asset.
