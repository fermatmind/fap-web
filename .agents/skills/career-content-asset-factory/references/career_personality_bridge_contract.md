# Career-Personality Bridge Contract

Bridge assets connect career content with assessment experiences.

## Purpose

- Help readers decide what to test next.
- Link career interest, work style, and page CTAs.
- Preserve boundaries between occupational facts and self-assessment outputs.

## Required Inputs

- PASS career identity and work-activities baselines.
- PASS fit block when available.
- PASS personality/test result assets for the target framework.

## Forbidden

- Claiming a type guarantees career fit.
- Claiming a test predicts income, admission, employment, or personal worth.
- Inferring personality traits from a career title alone.

## Gate

Bridge gate must check source traceability, CTA quality, locale independence, no diagnosis claims, and no deterministic match language.
