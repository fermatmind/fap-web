# Agent Boundary Matrix

| Agent | Owns | Must Not Do |
| --- | --- | --- |
| Career content orchestrator | global state, manifests, freeze, import readiness, next goal | write occupational facts |
| Identity block | definitions, title cleanup, official boundaries, codes | infer salary, fit, or risk |
| Work activities block | tasks, tools, stakeholders, environments | make salary/AI/fit claims |
| Salary block | salary evidence, estimates, salary reader assets | treat CN salary as official wage |
| Risk/future block | AI impact and future risk assets | predict job loss or personal outcomes |
| Fit block | RIASEC/personality/work-style fit signals | diagnose or guarantee success |
| Skills/entry block | skills, credentials, entry preparation | invent regional requirements |
| Adjacent comparison block | transfer paths and related careers | use title similarity as proof |
| Page assembly block | compose PASS assets into page display | generate new facts |
| SEO/GEO agent | sitemap, llms, schema, search release | bypass content/import approvals |
| Personality/test agents | test/result/profile assets | rewrite career facts |
| Release guard | deploy/import approval and post-import QA | approve content quality alone |
