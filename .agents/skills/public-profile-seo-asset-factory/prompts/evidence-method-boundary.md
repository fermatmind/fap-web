# Evidence And Method Boundary Prompt

Create or review evidence notes and method-boundary language.

## Requirements

- Separate academic, competitor, internal product, and inference evidence.
- Academic claims need DOI/URL, access date, and claim mapping.
- Competitor evidence can guide module gaps, not copy.
- Internal evidence must cite repo path, API endpoint, or report path.
- Method boundary must state what the content is not for.

## Forbidden

- Clinical diagnosis claims.
- Hiring or employment screening claims.
- Deterministic life decision claims.
- Official MBTI/A-T equivalence claims.
- Big Five fixed type claims.
- Enneagram clinical or hiring basis claims.
