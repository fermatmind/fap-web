# MBTI Rules

## Mode

Use `existing_asset_enhancement`.

## Current Estate

MBTI already has 64 bilingual A/T variant URLs and A/T comparison pages. Treat it as an existing indexable estate.

## Allowed

- Audit.
- Repair.
- Enhancement.
- Metadata/FAQ/content QA.
- A/T comparison improvement.

## Forbidden

- Rebuild or regenerate the 64-page estate.
- Claim A/T is official MBTI.
- Replace backend CMS/Profile authority.
- Copy result report body content.
