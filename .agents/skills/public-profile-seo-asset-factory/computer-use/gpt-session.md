# GPT Session

## Use GPT For

- Bilingual independent drafting.
- Structured section copy.
- FAQ drafting.
- Method-boundary wording.
- Content package normalization.

## Required Prompt Packet

Use `prompts/gpt-content-package.md` or `prompts/gpt-bilingual-independent-draft.md`.

## Required Output

GPT must return structured content, not runtime files. The output must identify claims that require source support. If GPT cannot provide a source for a factual claim, mark it as inference.
