# Transcript Capture

## Required Ledger Fields

- `model`
- `session_id`
- `started_at`
- `ended_at`
- `prompt_packet_path`
- `output_artifact_path`
- `summary`
- `source_claims`
- `unsupported_claims`
- `qa_status`
- `approved_uses`

## Storage

Store only safe summaries and approved excerpts. Do not store cookies, tokens, private user data, or private result payloads.
