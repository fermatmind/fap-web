# Private Result Boundary

## Allowed

- Taxonomy terms.
- Neutral structure inspiration.
- Generic non-user-specific phrasing.

## Forbidden

- Private report body copy.
- Score, percentile, close-call, or diagnostic internals.
- User-specific payloads.
- Report engine internals.
- Public copy like "your result this time" or "你这次结果".
- Paid-only module content.

Fail immediately on leakage.
