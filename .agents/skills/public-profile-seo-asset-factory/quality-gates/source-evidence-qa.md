# Source Evidence QA

## Required

- Academic claims cite DOI, URL, or verifiable official source.
- Competitor observations cite URL and access date.
- Internal product claims cite repo path, report path, API path, or test path.
- Unsupported model claims are marked inference or removed.

## Fail Conditions

- Source cannot be verified.
- A competitor sentence is copied.
- A clinical, hiring, or deterministic claim lacks approved authority.
