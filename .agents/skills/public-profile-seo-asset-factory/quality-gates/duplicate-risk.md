# Duplicate And Template Risk

## Check

- Compare sibling assets for repeated headings, repeated examples, and swapped labels.
- Ensure high/low polarity pages use different examples and boundaries.
- Ensure Enneagram core types are not nine copies with type numbers changed.
- Ensure MBTI enhancements target existing gaps instead of recreating boilerplate.

## Fail

Fail when an asset would look like programmatic doorway content.
