# 短视频脚本

## 30 秒版

开头 3 秒：选专业父母不同意，别先吵谁对。

正文：先拿一张表，把每个专业分成三步查。第一步看硬条件：位次、选科、招生计划、体检和单科限制。第二步看课程红线：四年到底学什么，你会不会长期抗拒。第三步用霍兰德兴趣结果，把兴趣转成任务问题：我愿不愿意长期做这些任务？

结尾：最后每个专业只进三栏：保留、待验证、暂时排除。测试不是志愿系统，只帮你把问题问清楚。

## 60 秒版

开头：很多家庭填志愿不是缺资料，而是父母和孩子意见不一致。父母说稳定，孩子说不喜欢，然后越聊越僵。

正文：换一个方法。不要先争谁更懂未来，先填一张专业证据表。第一列写专业，第二列写父母担心，第三列写孩子抗拒，后面写已验证事实、未验证假设、下一步验证和当前状态。

硬条件先过：位次、选科、招生计划、体检限制、单科要求、学费和城市。硬条件没过，先暂时排除。硬条件过了，再看课程红线。比如计算机要面对数学、代码、算法和项目；新闻传播要面对采访、写作、修改和实习。

霍兰德 / RIASEC 可以用，但不要把它当专业答案。它更适合问：我愿不愿意长期分析、表达、实操、帮助别人、组织资源或维护秩序？

结尾：每个专业只给三种状态：保留、待验证、暂时排除。正式填报还是看省考试院、招生章程和学校官方信息。
