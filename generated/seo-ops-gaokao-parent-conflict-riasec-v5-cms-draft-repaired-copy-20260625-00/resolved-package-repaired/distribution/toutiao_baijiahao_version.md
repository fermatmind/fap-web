# 百家号 / 今日头条版本摘要

高考志愿选专业时，如果父母和孩子意见不一致，最不建议的做法是继续围绕“热门”“稳定”“喜欢”争论。更稳妥的方式，是把每个专业放进一张共同证据表：先查位次、选科要求、招生计划、体检限制、单科要求等硬条件，再查课程、培养方案、真实任务和访谈信息，最后再用霍兰德 / RIASEC 职业兴趣测试辅助生成验证问题。

文章核心方法是三栏状态法：保留、待验证、暂时排除。硬条件过关、课程不明显抗拒、兴趣任务有证据，才进入保留；信息不足的专业放待验证；明显不满足条件或长期抗拒核心课程的专业，先暂时排除。

FermatMind 不是官方志愿填报系统，测试也不能替代省考试院规则、招生章程、选科要求和学校官方信息。测试更适合做探索起点，帮助家庭把分歧转成可验证的问题。
