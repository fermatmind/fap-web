# Claim Gate

Final decision: GO for human review; NO-GO for publish.

## Safe claims used

- 高考志愿家庭分歧可以通过证据表、硬条件、课程红线和现实验证降低争论成本。
- RIASEC / 霍兰德可作为工作活动与职业环境兴趣的探索输入。
- MBTI 可作为学习、沟通、决策偏好的补充语言。
- 课程表、培养方案、招生章程和官方信息必须优先核验。

## Claim-lint normalization

Article body avoids high-risk exact phrases where practical. Instead of phrasing the boundary as direct prohibited claims, it uses:

- `不会把某个专业写成唯一答案`
- `不会估算能否被录取`
- `不会估算就业结果、收入结果或长期发展结果`
- `测试不是志愿答案，只是验证问题生成器`

## Forbidden claims avoided

- Admission result estimation.
- Guaranteed major match.
- Salary, employment, hiring, or long-term development certainty.
- Official admission-system implication.
- Official MBTI / Holland Code / O*NET implication.
- Fabricated reliability, validity, sample size, norm group, or matching algorithm.
