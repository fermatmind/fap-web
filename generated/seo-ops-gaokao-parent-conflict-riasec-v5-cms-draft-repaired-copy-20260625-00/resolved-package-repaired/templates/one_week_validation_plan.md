# 一周验证任务表

| 日期 | 验证任务 | 输出物 | 判断问题 | 完成情况 |
|---|---|---|---|---|
| 第 1 天 | 查招生章程、选科、招生计划、体检限制 | 硬条件记录 | 能不能进入讨论区间？ |  |
| 第 2 天 | 查培养方案和核心课程 | 课程红线清单 | 我长期抗拒哪些课程？ |  |
| 第 3 天 | 看 1 节公开课或入门讲座 | 体验记录 | 我是更有兴趣，还是更想逃开？ |  |
| 第 4 天 | 做 1 个微型任务 | 作品/笔记/代码/案例摘要 | 我是否愿意继续做第二次？ |  |
| 第 5 天 | 访谈 1-2 个在读学生或从业者 | 访谈摘要 | 真实体验和想象差在哪？ |  |
| 第 6 天 | 回看 RIASEC / MBTI 结果 | 验证问题清单 | 测试是否帮助我提出更具体问题？ |  |
| 第 7 天 | 家庭复盘 | 保留 / 待验证 / 暂时排除 | 这个专业下一步是什么？ |  |
