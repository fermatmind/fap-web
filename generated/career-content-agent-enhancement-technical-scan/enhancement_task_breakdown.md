# Career Content Agent Enhancement Task Breakdown

## Lane 1: SEO/GEO Query Intent Optimization

Purpose: create candidate-only query intent assets for each career page. Do not change runtime SEO, sitemap, llms, canonical, noindex, robots, JSON-LD, Search Channel, or CMS.

Tasks:
1. Read-only query intent audit for 1046 slugs.
2. Candidate projection: primary intent, secondary intent, snippet candidate, FAQ candidate, anchor text candidate.
3. SEO/GEO release gate design.
4. Optional 50-slug staging preview only after backend authority exists.

## Lane 2: Internal Links and Adjacent Career Graph

Purpose: improve career-to-career navigation using completed adjacent-comparison baseline and backend authority.

Tasks:
1. Audit current next-step links and companion links.
2. Generate candidate graph: adjacent jobs, family hub, upstream/downstream paths, assessment entry.
3. Backend reader-safe projection PR.
4. Frontend rendering PR.
5. 50-slug and then 1046 QA.

## Lane 3: RIASEC / Big Five / MBTI Conversion Bridge

Purpose: connect career pages to tests without deterministic career-match claims.

Tasks:
1. Extend RIASEC bridge contract to Big Five and MBTI.
2. Generate candidate CTA bridge assets using public-safe inputs only.
3. Claim safety gate: no best-career, guaranteed-fit, salary/success/hiring/admissions/performance prediction.
4. Backend preview projection.
5. Frontend conversion module with attribution.

## Lane 4: High-value Career Manual Editorial Upgrade

Purpose: upgrade a selected cohort instead of rewriting all 1046.

Tasks:
1. Select 50-150 high-value careers by search, conversion, risk, and business priority.
2. Create block-level editorial briefs.
3. Repair source-backed block assets in batches.
4. Regenerate page assembly for upgraded cohort only.
5. Staging preview/editorial QA before import.
