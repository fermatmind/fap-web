# Career Salary Asset Factory Runbook

1. Build or select a 50-row batch manifest.
2. Generate CN/US/UK/EU collection plans. Use Browser/Chrome only for dynamic or logged-in pages.
3. Produce evidence ledger only from captured evidence.
4. Validate and audit evidence. Repair up to 3 loops.
5. If evidence is not PASS, stop.
6. Compute estimates only from PASS evidence.
7. Audit estimates. If not PASS, stop.
8. Generate assets only from PASS evidence plus PASS estimates.
9. Audit assets. If PASS, promote/freeze and record SHA.

Default batch size is 50. Scale only according to `references/batch_gate_rules.md`.
