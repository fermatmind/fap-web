# IQ Method Pages zh-CN v0.2

Status: `draft_review_only`

This package contains seven zh-CN SEO/GEO methodology, boundary, and trust-layer pages for FermatMind IQ V1. It is intended for human review and later CMS dry-run import only.

## Scope

- Product framing: `IQ 风格推理测试`
- Test route: `/zh/tests/iq-test-intelligence-quotient-assessment`
- Test facts: FermatMind original 30 items, about 20 minutes, backend-private scoring.
- Allowed public result language: raw score, correct rate, completion time, dimension performance, stability note, and current reasoning-task performance.
- Current high-level dimensions: 视觉空间洞察, 视觉空间模式推理, 数字规律推理.
- No item generation, answer generation, answer-key disclosure, scoring-rule disclosure, CMS write, publish, schema activation, sitemap/llms mutation, or search submission.
