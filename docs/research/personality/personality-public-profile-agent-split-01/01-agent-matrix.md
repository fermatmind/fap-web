# Public Profile Agent Matrix

## Agent Responsibilities

| Agent | Primary Responsibility | Current Scope |
| --- | --- | --- |
| Content Orchestrator Agent | Routes work and enforces PR/gate sequencing | All public profile frameworks |
| MBTI64 Public Personality Agent | Existing MBTI64 estate improvement | 64 A/T variants and 32 comparisons |
| Big Five Public Personality Agent | Big Five public profile packages | 5 domains, 10 poles, 30 facets |
| Enneagram Public Personality Agent | Enneagram public placeholder upgrades | Hub, 3 centers, 9 core types |
| SEO Projection QA Agent | SERP, URL Truth, sitemap, llms, Search Queue readiness | Readiness and report artifacts |
| Editorial Claim QA Agent | Claim, trademark, duplicate, and private-result checks | Pre-publish gates |
| Release Guard Agent | Dry-run/write/publish/search separation | All release actions |

## Workflow

1. Orchestrator receives the target framework and task stage.
2. Framework agent creates or audits a structured asset packet.
3. Editorial Claim QA checks trademark, method, duplicate, and private-result boundaries.
4. SEO Projection QA checks metadata, canonical, discovery surface, URL Truth, and queue readiness.
5. Release Guard decides whether the next step is still read-only, draft write, promotion, index surface, Search Queue, or observation.

## Scope Boundary

This PR defines the agent contracts only. It does not run MBTI64 expansion, produce Big Five or Enneagram content packages, import CMS content, publish pages, or submit URLs to search providers.
