# Automation Boundary Audit

## Preserved Boundaries

- No runtime code.
- No fap-api code.
- No sitemap/llms/indexability change.
- No publish.
- No formal production content package.
- No live GPT/Gemini drafting.

## Private Result Boundary

The skill states that result pages may inspire taxonomy or structure only. It forbids private report body copy, score/percentile/user-specific payloads, report engine internals, and public copy framed as a user's specific result.

## Repository Authority

The skill states fap-api is schema/import/API/content authority. fap-web is renderer, SEO, sitemap/llms gate, and PR-train owner.
