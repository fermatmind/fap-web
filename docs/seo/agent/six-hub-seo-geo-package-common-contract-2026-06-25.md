# Six Hub SEO/GEO Package Common Contract

Task: `SIX-HUB-SEO-GEO-PACKAGE-COMMON-CONTRACT-01`

Verdict: `READY_FOR_SEO_GEO_PACKAGE_PLANNING`

This common contract defines the shared docs/contracts vocabulary for the 12 public assessment Hub pages. It converts the completed `SIX-HUB-SEO-GEO-PACKAGE-PLANNING-SCAN-01` into a package-planning contract only.

It does not authorize CMS dry-run, CMS write, publish, runtime copy repair, frontend implementation, backend API repair, search submission, provider execution, deploy, private result access, generated pages, or SEO artifact mutation.
