# Core Decision Domain v1

Runtime behavior changed: no.

This artifact defines the Phase 4A governance overlay for the first three
FermatMind decision domains. It is an artifact, generated JSON, and contract
layer only. It does not add domain hub pages, public routes, visible copy,
runtime CTA behavior, recommendation runtime, profile writes, SEO/GEO
exposure, freemium runtime, report entitlement, checkout/payment behavior, or
scoring behavior.

## Source Artifacts

The overlay inherits UASP, Semantic Claim Boundary, public runtime authority,
Evidence Container, freemium, and graph governance artifacts. Missing source
artifacts are recorded in the generated JSON residual risk section rather than
fabricated.

## Domains

### self_understanding

- User problem: 我是谁？我的核心倾向是什么？
- Allowed scales: MBTI, BIG5_OCEAN, ENNEAGRAM
- Blocked scales: RIASEC as primary self-identity authority, future clinical
  or ability scales until separate approval.
- Boundary: no determinism, no diagnosis, no destiny framing, no generic
  personality-entertainment framing.
- Runtime status: artifact_only.
- Readiness status: ready_for_domain_v1.

### career_decision

- User problem: 我适合什么职业方向？下一步该探索什么？
- Allowed scales: RIASEC, MBTI, BIG5_OCEAN, Career Graph.
- Blocked scales: ENNEAGRAM for career decision mainline, future ability tests,
  and future emotional/clinical signals.
- Boundary: career direction support only; no precise recommender, no best-career
  prediction, no success or placement guarantee, no Big Five/RIASEC career
  matcher, no AI planning claim.
- Runtime status: artifact_only.
- Readiness status: partial.

### workstyle_decision

- User problem: 我适合怎样的工作方式？在团队里怎么发挥？
- Allowed scales: BIG5_OCEAN, MBTI, ENNEAGRAM.
- Blocked scales: RIASEC and future ability tests. Future DISC/EQ are future
  placeholders only.
- Boundary: no employment suitability, no workplace performance prediction, no
  HR screening claim, no Big Five career matching.
- Runtime status: artifact_only.
- Readiness status: partial.

## No Runtime Change Statement

This overlay is governance-only. Domain mapping does not grant recommendation,
profile, SEO/GEO, freemium, CTA, or report entitlement eligibility.
